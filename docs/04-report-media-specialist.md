# Team Report: Media Specialist

**Role:** Media Sourcing, Post-Production, Asset Preparation

## Scope of Work

Sourcing, selecting, processing, and preparing all visual media assets for the archive — 180 event posters, event photography across 180 slideshow galleries, and supplementary visual materials. Work encompassed archive research, image selection from historical documentation, color correction and grading, format conversion and optimization, file organization, and delivery pipeline.

## Deliverables

### 1. Poster Collection (180 events)

Sourcing of original poster files or high-quality scans for all 180 events spanning 2012–2025. Where original digital files were unavailable, coordination of physical poster scanning. Resolution verification and upscaling assessment. Cleanup of scan artifacts — dust, fold marks, color drift from aging.

### 2. Poster Post-Production

Color correction of all 180 posters to ensure consistent screen rendering across varying source quality — professional print files, phone photographs, scanned flyers, social media exports. White point normalization. Contrast adjustment for web display. Preservation of original design intent while ensuring readability at card-index thumbnail scale.

### 3. Poster Optimization

Conversion of all 180 posters to WebP format at optimized quality/size ratio. Resolution standardization for consistent grid display. File naming convention (c1.webp–c180.webp). Total poster directory: 37 MB across 180 files.

### 4. Event Photography — Selection

Review and selection of event photography from 14 years of documentation across multiple photographers and sources. Per-event curation: choosing representative images that capture the character of each event — performance moments, venue atmosphere, audience, installation views. Balancing visual variety across the archive while maintaining documentary authenticity per art direction guidelines.

### 5. Event Photography — Post-Production

Color grading of selected photographs for visual consistency within each event gallery and across the archive. Exposure correction for venue photography — low-light performance documentation, mixed lighting conditions. Cropping and composition adjustment where necessary. Removal of distracting artifacts.

### 6. Photography Optimization

Export of all selected photographs at web-optimized resolution. Organization into 180 event folders (c001–c180) with consistent naming. Format selection — JPG for photographs, PNG where transparency required. Total slideshow directory: 48 MB across 180 folders.

### 7. Supplementary Assets

Preparation of the Goethe-Institut logo (cropped, transparency-optimized PNG). Sourcing and delivery of the 3D model file (kopalyny.usdz, 5.5 MB) for the home page. Favicon preparation.

### 8. Asset Pipeline

Organization of the complete media directory structure. CDN upload and URL verification for all poster and slideshow assets. Cross-referencing media files with event data records to ensure correct ID mapping. Quality verification pass at final integration.
